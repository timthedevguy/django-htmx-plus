# AI Usage

I personally wrote the base feature set and used AI to tweak and improve.  The documentation and docstrings are AI
generated.

AI was used to start/flesh out the following:

* Support for Browswer History using hx-push-url
* Documentation
* Better docstrings
* All unit tests

Claude Sonnet is my model of choice as I don't let AI write huge chunks of code, mostly reviewing my code and offering 
suggestions.
